# DriveClubFS

Unpacks files from DriveClub. (.ndx + .dat).

Supports:
* 1.28
* NPXX51272 - Alpha/Proto Build

Does NOT currently support:
* 1.00


> [!WARNING]
> This tool does not deal with .rpk resource packs.

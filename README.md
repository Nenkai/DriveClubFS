# DriveClubFS

Unpacks files from DriveClub. (.ndx + .dat).

Also supports extracting binary resources, xmls and textures (DriveClub) from `.rpk` resource packs.

Supports:
* 1.00
* 1.28
* NPXX51272 - Alpha/Proto Build
